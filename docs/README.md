# Zen Prompt Data (Static)

Welcome to the static Data for **Zen Prompt**: aesthetic inspiration for your shell.

## 🚀 Data Access

We provide two tiers of data in both SQLite and JSON formats.

### 🗄️ SQLite Data (Recommended for CLI)

Optimized with **FTS5 Full-Text Search** indexing.

- **[quotes.db](./data/sqlite/quotes.db)**: The complete optimized collection with FTS5.
- **[quotes-small.db](./data/sqlite/quotes-small.db)**: A smaller subset (filtered by 'buddhism' tag) for instant availability.

### 📄 JSON Data

Standard JSON exports for web integration.

- **[quotes.json](./data/json/quotes.json)**: All quotes in a single array.
- **[quotes-small.json](./data/json/quotes-small.json)**: A subset of quotes with the 'buddhism' tag.

### 📊 CSV API
CSV exports for spreadsheets and data analysis.

- **[quotes.csv](./data/csv/quotes.csv)**: All quotes.
- **[quotes-small.csv](./data/csv/quotes-small.csv)**: A subset of quotes with the 'buddhism' tag.

## 💻 Using the CLI Client

The easiest way to interact with this hosted Data from your terminal is to install the [Zen Prompt](https://github.com/anhldbk/zen-prompt) CLI client.

### Installation

```bash
pip install zen-prompt
```

### 1. Sync the Data

Synchronize your local cache with the remote static website:

```bash
zen-prompt sync https://anhldbk.github.io/zen-prompt/data/
```

### 2. Get a Random Quote

Instantly display a random quote from the cache with the default monochrome photo:

```bash
zen-prompt random

# Disable the default photo
zen-prompt random --no-photo

# Pick only shorter quotes
zen-prompt random --quote-max-words 20 --quote-max-chars 140

# Pick a random hero photo from a topic directory
zen-prompt random --photo topic@monochrome

# Limit image height and width in the terminal
zen-prompt random --photo topic@monochrome --photo-max-height 10 --photo-max-width 80

# Put the photo beside the quote
zen-prompt random --photo topic@monochrome --photo-layout table

# Wrap long quotes to a fixed width
zen-prompt random --quote-width 60
```

### 3. Search by Keyword

Perform a lightning-fast search across all quotes:

```bash
zen-prompt search "success and happiness" --limit 5
```
